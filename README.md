# PlantCare

Projeto de pesquisa em tecnologia embarcada inteligente para o setor rural, com diagnóstico de doenças em plantas a partir de fotos de folhas.

## Estrutura do repositório

- `/mobile` — App Flutter (protótipo a ser integrado)
- `/backend` — API Java Spring Boot + Spring Data JPA
- `/docs` — Documentação e imagens de referência (incluindo o protótipo `PlantCareProt.png`)
